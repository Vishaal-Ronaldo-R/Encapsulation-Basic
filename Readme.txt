# Encapsulation-Basic

This project demonstrates the concept of encapsulation in C#.

## Description

Encapsulation is one of the fundamental principles of object-oriented programming. It involves bundling data and methods that operate on that data within a single unit, typically a class, and restricting access to some of the object's components.

## Features

- Example classes showing encapsulation
- Public, private, and protected members

## Installation

Clone the repository:

```bash
git clone https://github.com/Vishaal-Ronaldo-R/Encapsulation-Basic
```

Run the program to see encapsulation in action.
